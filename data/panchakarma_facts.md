| パンチャカルマ (五業)。 | 体系的な浄化療法。余分なドーシャやアーマを浄化し、病気の根を取り除く。心身を浄化し、健康増進、若返りを目指す。 #前処置 #中心処置 #後処置。 | 増悪したドーシャ、アーマの蓄積がある病態。健康人の健康増進、老化制御（ラサーヤナの中心）。 | アーマが蓄積している人（一旦熟成・消化してからでないと排泄できない）。急性症状で発熱がある人。老人、10歳以下（ヨーガの浄化法を含む）。 | アーマ・パーチャナ（消化剤法）を3〜10日間。スネーハナ（油剤法）、スウェーダナ（発汗法）により毒素を溶解・排泄しやすくする。 | 前処置で消化管内や鼻腔内、皮下に導き出された毒素を、5つの方法で体外へ排除する。 | サムサルジャナ（食餌法）：軽くて消化しやすい食事。ラサーヤナ（強壮剤法）：不老長寿薬を処方。中心処置期間の2〜3倍の期間をかけて徐々に元の生活に戻す。 |
| ヴァマナ (催吐法)。 | 胃内や胸腔に過剰になったドーシャ（主にカパ）やアーマを体外へ排出し、粘液や欝滞を除去する。悪化したカパを胃に集め、そこから吐き出す。 #前処置 #中心処置 #後処置 #カパ。 | 過剰なカパ。カパ性疾患に最も有効な浄化療法。 | - | 前処置（アーマ・パーチャナ、スネーハナ、スウェーダナ）を体系的に行う。 | 口から催吐させる。事前に計画された処理によって、悪化したカパを胃の中に集め、そこから吐き出す。 | 治療後、後処置を行う。 |
| ヴィレーチャナ (潟下法)。 | 小腸からの毒素を排出。過剰になったピッタやアーマを排除する。全身の悪化したピッタを腸に集め、肛門から排泄させる。 #前処置 #中心処置 #後処置 #ピッタ。 | 過剰なピッタ。ピッタ増悪時には欠かせない。 | - | スネーハ・パーナ（油剤飲用法）を3〜5日行った後に実施。 | 下剤（ヒマシ油など）を内服させる。 | 後処置として3日〜7日ほど生活指導を行う。 |
| バスティ (浣腸法)。 | 主に大腸から過剰になったヴァータやアーマ、マラを排出させる。病気の根っこを取り除く方法としてパンチャカルマには必須。 #前処置 #中心処置 #ヴァータ #禁忌。 | 過剰なヴァータ。適応と不適応（禁忌）をよく熟知して行うことが大切。 | - | 前処置（油剤法、発汗法）により、中心処置の負荷でヴァータが乱れるのを防ぐ。 | 肛門を通り大腸から排出させる。ニルーハ・バスティ（貧油性：浄化作用が強い）とアヌヴァーサナ・バスティ（油性：ヴァータ鎮静化）の2種類。通常、中心処置として毎日行う。 | - |
| ナスヤ (経鼻法)。 | 鼻、喉、副鼻腔、頭部を浄化する。過剰になったカパやアーマを排除する。 #中心処置 #カパ #推奨。 | 過剰なカパ。頭痛、鼻乾燥、鼻閉塞、かすれ声、偏頭痛、アレルギー性鼻炎、目や耳の疾患。カパ増悪時の中心処置として適当。 | - | - | 鼻に油などを点鼻する。薬用ギー（ブラーミーギー、アヌ・タイラ）を使用。 | - |
| ラクタ・モークシャナ (瀉血法)。 | 皮膚から直接毒素を排泄。ラクタ（血液）のマラとしてのピッタや、流血中のアーマを排除し、血液を浄化する。 #中心処置 #ヴァータ #ピッタ #推奨。 | 汚濁した血液。ラクタの乱れ。蕁麻疹、発疹、湿疹、ニキビ、白斑、慢性の皮膚病、肝腫大、痛風、肋膜炎、心筋炎など。ピッタやヴァータ増大時で排泄が必要な場合。 | - | - | 皮膚や血管から排出させる。ヒルに血を吸わせる方法や、カッピング（吸角を使用）や放血が適している。ピッタを浄化する中心処置としても良い。 | - |
